Deepfake

term_id: deepfake

Category: ethics_safety

Definition

Deepfakes sind hyperrealistische Audio- oder Videomanipulationen, die mit generativen adversariellen Netzwerken (GANs) oder Autoencodern erstellt werden. Sie werfen erhebliche ethische Bedenken hinsichtlich Fehlinformationen, Privats…

Summary

Synthetische Medien, bei denen das Erscheinungsbild einer Person mithilfe von künstlicher Intelligenz und Deep-Learning-Techniken durch das einer anderen ersetzt wird.

Key Concepts

  • Generative Adversarial Networks
  • Medienmanipulation
  • Fehlinformation
  • Forensische Erkennung

Use Cases

  • Erkennung betrügerischer Identitätsüberprüfungen
  • Analyse politischer Desinformationskampagnen
  • Entwicklung von Algorithmen zur Deepfake-Erkennung
