Federated Learning

term_id: federated_learning

Category: ethics_safety

Definition

Federated Learning ermöglicht es Organisationen, KI-Modelle kollaborativ zu trainieren, ohne sensible Rohdaten auszutauschen. Anstatt Informationen zu zentralisieren, wird das Modell an lokale Geräte gesendet, wo es dort lernt, und nur die Modellaktualisierungen werden zurückgesendet, um die Privatsphäre der Nutzer zu schützen.

Summary

Federated Learning ist ein verteilter Machine-Learning-Ansatz, der Modelle über dezentrale Geräte hinweg trainiert, wobei die Daten lokal bleiben.

Key Concepts

  • Datenschutz
  • Verteiltes Training
  • Modellaggregation
  • Edge Computing

Use Cases

  • Prädiktive Texttastaturen
  • Zusammenarbeit bei medizinischen Diagnosen
  • Erkennung von Finanzbetrug
