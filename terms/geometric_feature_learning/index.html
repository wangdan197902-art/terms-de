Geometrisches Feature-Learning

term_id: geometric_feature_learning

Category: training_techniques

Definition

Geometrisches Feature-Learning konzentriert sich auf die Verarbeitung von Daten, die nicht-euklidische Strukturen besitzen, wie soziale Netzwerke, Molekülgraphen oder 3D-Meshes. Techniken wie Graph Neural Networks (GNNs) und…

Summary

Eine Technik im Deep Learning, die Merkmale aus Daten mit inhärenten geometrischen Strukturen, wie Graphen oder Mannigfaltigkeiten, unter Verwendung spezialisierter neuronaler Netze extrahiert.

Key Concepts

  • Graph Neural Networks
  • Mannigfaltigkeitslernen
  • Erhaltung der Symmetrie
  • Topologische Datenanalyse

Use Cases

  • Arzneimittelforschung und Molekülmodellierung
  • Analyse sozialer Netzwerke
  • Erkennung von 3D-Objekten

Related Terms

  • Graph Convolutional Network (GCN)
  • Äquivarianz (Invariantes Verhalten unter Transformationen)
  • Mannigfaltigkeits-Hypothese (Annahme niedrigdimensionaler Struktur)
  • Relationales Lernen (Lernen aus Beziehungen)
