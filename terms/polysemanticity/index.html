Polysemantizität

term_id: polysemanticity

Category: basic_concepts

Definition

Polysemantizität ist eine Eigenschaft, die in tiefen neuronalen Netzen beobachtet wird, insbesondere in Transformern. Dabei kann ein einzelnes Neuron als Reaktion auf mehrere unzusammenhängende oder semantisch verschiedene Merkmale aktiviert werden.

Summary

Das Phänomen, bei dem einzelne Neuronen in neuronalen Netzen auf mehrere unterschiedliche Konzepte reagieren.

Key Concepts

  • Interpretierbarkeit von Neuronen
  • Sparsame Autoencoder
  • Entkopplung von Merkmalen
  • Transformer-Architektur

Use Cases

  • Forschung zur mechanistischen Interpretierbarkeit
  • Debugging des Modellverhaltens
  • Verbesserung der Transparenz und Sicherheit von Modellen

Related Terms

  • Monosemantizität (Jedes Neuron reagiert nur auf ein Konzept)
  • Interpretierbarkeit (Nachvollziehbarkeit von KI-Entscheidungen)
  • Sparsame Kodierung (Datenrepräsentation mit wenigen aktiven Einheiten)
  • Neurale Sonden (Methoden zur Analyse neuronaler Aktivität)
