Edge Computing

term_id: edge_computing

Category: engineering_practice

Definition

Edge Computing adressiert die Latenz- und Bandbreitenbeschränkungen cloudbasierter Architekturen, indem Daten dort verarbeitet werden, wo sie erzeugt werden, wie bei IoT-Geräten, Sensoren oder lokalen Gateways. Im KI-Kontext

Summary

Edge Computing ist ein verteiltes Paradigma der Datenverarbeitung, das Rechenleistung und Datenspeicher näher an die Datenquellen bringt, anstatt sich ausschließlich auf zentralisierte Cloud-Server zu verlassen.

Key Concepts

  • Niedrige Latenz
  • Verteilte Verarbeitung
  • IoT-Integration
  • Modellkomprimierung

Use Cases

  • Navigation autonomer Fahrzeuge
  • Intelligente Kameraüberwachung
  • Prädiktive Wartung in der Industrie
