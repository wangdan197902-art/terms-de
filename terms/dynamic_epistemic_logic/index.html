Dynamische epistemische Logik

term_id: dynamic_epistemic_logic

Category: engineering_practice

Definition

Die Dynamische Epistemische Logik (DEL) erweitert die modale Logik, um zu modellieren, wie sich Wissen entwickelt, wenn Agenten neue Informationen erhalten. Sie bietet Werkzeuge zur Analyse von Multi-Agenten-Systemen, in denen Überzeugungen aufgrund öffentlicher…

Summary

Ein formaler Rahmen zur Begründung von Änderungen im Wissenszustand von Agenten infolge von Informationsaktualisierungen oder Ereignissen.

Key Concepts

  • Wissensaktualisierung
  • Informationsfluss

  • Verifikation von Sicherheitsprotokollen
  • Modellierung der Kommunikation in Roboterschwärmen
  • Analyse von spieltheoretischen Szenarien
