Zero-Shot-Lernen

term_id: zero_shot_learning

Category: training_techniques

Definition

Zero-Shot-Lernen ermöglicht es einem Machine-Learning-Modell, Instanzen von Klassen zu klassifizieren, die nicht im Trainingsdatensatz vorhanden waren. Statt auf bespielte Beispiele für jede mögliche Klasse zu vertrauen, nutzt das Modell semantische Beschreibungen.

Summary

Zero-Shot-Lernen ist eine Technik, bei der ein Modell Aufgaben für Klassen ausführt, die es während des Trainings nie gesehen hat, indem es semantische Assoziationen nutzt.

Key Concepts

  • Verallgemeinerung
  • Semantische Einbettungen
  • Attributlernen
  • Transferlernen

Use Cases

  • Klassifizierung seltener Tierarten
  • Identifizierung neuer Produktkategorien
  • Anpassung an neue Bedrohungen ohne Nachschulung

Related Terms

  • Few-Shot-Lernen (Lernen mit wenigen Beispielen)
  • Transferlernen (Wissenübertragung zwischen Aufgaben)
  • Semantische Einbettung (Vektorrepräsentation von Bedeutung)
  • Verallgemeinerung (Fähigkeit, auf neue Daten zu passen)
