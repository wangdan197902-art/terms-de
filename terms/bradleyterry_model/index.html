Bradley-Terry-Modell

term_id: bradleyterry_model

Category: basic_concepts

Definition

Das Bradley-Terry-Modell ist ein probabilistisches Modell, das weit verbreitet in der Psychometrie und im maschinellen Lernen eingesetzt wird, um paarweise Vergleiche zu verarbeiten. Es weist jedem Item einen latenten Wert zu und berechnet die Wahrscheinlichkeit

Summary

Ein statistisches Modell zur Analyse von paarweisen Vergleichsdaten, das die Wahrscheinlichkeit schätzt, dass ein Item einem anderen vorgezogen wird.

Key Concepts

  • Paarweiser Vergleich
  • Rangordnungssysteme
  • Logistische Regression
  • Präferenzlernen

Use Cases

  • Algorithmen zur Sport-Rangliste
  • Analyse von A/B-Tests
  • Bestärkendes Lernen aus menschlichem Feedback (RLHF)

Related Terms

  • Elo-Bewertung (System zur Berechnung der relativen Stärke von Spielern)
  • Plackett-Luce-Modell (Statistisches Modell für Rangfolgen)
  • Isotone Regression (Regressionsmethode unter Monotoniebedingungen)
  • RLHF (Bestärkendes Lernen aus menschlichem Feedback)
