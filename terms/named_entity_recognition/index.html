Named Entity Recognition

term_id: named_entity_recognition

Category: application_paradigms

Definition

Named Entity Recognition (NER) ist eine Teilaufgabe der Informationsextraktion, die benannte Entitäten in Texten lokalisiert und in vordefinierte Kategorien wie Personennamen, Organisationen, Standorte, medizinische Begriffe oder Datumsangaben einordnet.

Summary

Eine Aufgabe im Natural Language Processing (NLP), die Schlüsselinformationen in Text identifiziert und vordefinierten Kategorien zuordnet.

Key Concepts

  • Entitätstypisierung
  • Token-Klassifizierung
  • Sequenzbeschriftung
  • BiLSTM-CRF-Architekturen

Use Cases

  • Lebenslauf-Parsing
  • Erkennung von Kundenanliegen im Support
  • Analyse von Patientenakten
